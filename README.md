## 환불 규정 요약

### 기간권
| 상품명 | 상품금액 | 1주차 | 2주차 | 3주차 | 4주차~ |
|--------|---------|-------|-------|-------|--------|
| 2주권 | 55,000원 | 22,000원 | 불가 | — | — |
| 4주권 | 119,000원 | 76,300원 | 43,600원 | 21,800원 | 불가 |
| 고정석 4주권 | 179,000원 | 125,300원 | 71,600원 | 35,800원 | 불가 |
| 관리고정석 4주권 | 229,000원 | 160,300원 | 91,600원 | 45,800원 | 불가 |
| 오픈행사 4주권 | 79,000원 | 55,300원 | 31,600원 | 15,800원 | 불가 |

### 시간권
- **환불금액 = 상품금액 − (이용시간 × 1,500원)**
- 음수가 되면 0원 처리
- 구매 후 28일 초과 시 환불 불가

| 상품명 | 상품금액 |
|--------|---------|
| 100시간권 | 95,000원 |
| 200시간권 | 180,000원 |
| 오픈100시간권 | 69,000원 |

### 환불 불가 상품
- **당일권**: 환불 불가
- **스터디룸**: 환불 불가 (예약 일정 변경만 가능)

### 확장 기간권 (2~20주권)
전체 기간을 4분기로 균등 분할하여 비율 적용:
- 1분기: 70% 환불
- 2분기: 40% 환불
- 3분기: 20% 환불
- 4분기: 환불 불가

## API 사용 예시

```python
from refund_calculator import calculate_refund, calculate_extended_period_refund

# 기간권
result = calculate_refund("기간권", "4주권", 10)
# → {"환불금액": 43600, "환불가능": True, "사유": "2주차 사용 → 43,600원 환불", ...}

# 시간권
result = calculate_refund("시간권", "100시간권", 20)
# → {"환불금액": 65000, "환불가능": True, ...}

# 확장 기간권 (8주권, 200,000원, 15일 사용)
result = calculate_extended_period_refund(8, 200_000, 15)
# → {"환불금액": 80000, "환불비율": "40%", ...}
```
